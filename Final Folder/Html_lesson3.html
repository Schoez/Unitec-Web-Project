HTML Editing

Learn HTML using Notepad

Web pages are usually created and edited using professional HTML editors.

However, when learning HTML we would recommend a simple text editor like Notepad or Notepad++.

We believe that using a simple text editor is a really good way to start learning HTML.

You can follow the steps below to create your first web page with Notepad or Notepad++.



Step 1: Getting started with Notepad

On Windows 7 or earlier:

Open "Start > Programs > Accessories > Notepad" after doing this notepad should open.

On Windows 8 or Newer:

Open the start screen then type "notepad" in the search and press enter. After doing this notepad should open.
