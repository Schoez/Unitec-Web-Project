How to change Sizes and fonts



The lesson will be describing how to change sizes for

  1. Images
  2. Texts


1. How to change the size for the Images.


You can set height and width with

  1. A number of Px(pixel)
  2. A number
  3. A Percentage.

In External and Internal CSS ,height and width code should look like


Img { width: XY px (or XY or XY%) ; height: XY px (or XY or XY%) ; }


In Inline ,height and width code should look like

<

Img src="image name" Width="XY px(or XY or XY%) Height=" px(or XY or XY%)"

>

XY means any Number






2. How to size Texts


You can set the font size with

  • A number of a Font size

In External and Internal CSS , code should look like:


p {font-size:XY ; }


In Inline code should look like:


<

a style="font-size:XY"

>

XY means any Number


3.Chaging font style


changing Font style' is very simular to changing size


In External and Internal CSS ,code should look like


p {font-style: Name of the style ; }


OR

p {font-weight: thickness of the font (e.g.bold or 900) ; }


In Inline, code should look like


<

p style="font-style: Name of the style"

>


OR

<

p style="font-wight: thickness of the font (e.g.bold or 900)"

>
