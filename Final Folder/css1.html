What is Css??

CSS(Cascading Style Sheets) is a way to set up how elements will be displayed .

This lesson will describe all 3 ways of using css

  • External css
  • Internal css
  • Inline css

And how to change colors with css.












1.External css



External css is a way of styling the webpage by setting styles in separate file(.css) and linking to the HTML web page.

1.How to link the css file

  • In the Head of html page. Type," Angle bracket link Href ="name of css file.css" rel="stylesheet" Angle bracket closed"

2.How to make external css file(.css)


  1. open notepad++
  2. type in css code
  3. SAVE as Casade Style Sheet file (*.css)


The advantages of External Style Sheets are:

  • the styles of html webpage can be controlled from one file(.css).
  • Classes can be created for use on multiple HTML element types in many documents.
  • In complex situations, selector and grouping methods can be used to apply styles.

The disadvantages of External Style Sheets are:

  • extra download is required to put style information for each document.
  • Until the external style sheet is loaded, it may not be possible to render the document.
  • For small number of style definitions wont be viable.


2.Internal css


Internal CSS is styling by adding CSS code in the HEAD section of the document.


Styling changes will apply to every specific element in the html file.
this is a solid option for one-page websites or when you cannot create external stylesheets for your project.

The advantages of internal Style Sheets are:

  • It can over write the codes from external css(stronger than external css)
  • You can use class and ID selectors
  • In complex situations, selector and grouping methods can be used to apply styles.

The disadvantages of Internal Style Sheets are:

  • increase the html pageâ€™s size and loading time.



3.Inline css


Inline CSS is applying a unique style to one HTML element at a time.


The advantages of Inline css are:

  • It over writes the codes from external css and internal css (stonger than the internal and external style sheet)
  • You can change one specific element with out effecting other elements

The disadvantages of Inline css are:

  • makes the html page messy.
  • increase the size of page and download time.
