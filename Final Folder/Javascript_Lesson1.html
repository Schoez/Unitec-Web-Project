What is Javascript ?

JavaScript is a dynamic computer programming language. It is lightweight and most commonly used as a part of web pages, whose implementations allow client-side script to interact with the user and make dynamic pages. It is an interpreted programming language with object-oriented capabilities.The programs in this language are called scripts. They can be written right in a web page’s HTML and run automatically as the page loads.

Introduction

JavaScript was created to add ‘life to webpages’. Just like HTML and CSS, JavaScript is written in plain text files with a .js extension.JavaScript has no relation to Java.Javascript is the collection of properties and methods consisting of set of definable characteristics that you can view and modify, and with which you can interact.Colors are good example of a property that, because of its nature, is easy to visualize. Properties that apply to HTML document can be directly manupulated by referring to the object and the property by name and then settigns its value. For example,the background color of a page can be changed by a concise statement: document.bgcolor="blue".

Methods are techiques that are used to perform actions involving objects and their properties. Javascript can manipulate the behaviour of a number of known objects- those based on HTML objects. One of the simplest method is the write method. It applies to the document object and causes a string of characters to the written in the HTML document. The simple statement document.write("Hello World!") displays Hello World onscreen in HTML when the browser encounters that statement as a part of a script.

How does Javascript work?

JavaScript is either embedded into a web page or else it’s included in a .js file. JavaScript is also a “client-side” language (rather than a “server-side” language), which is a fancy way of saying that it gets downloaded to site visitors’ computers, then processed.
