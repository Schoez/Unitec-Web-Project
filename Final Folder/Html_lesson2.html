Introduction

What is HTML?

  • HTML is an abbreviation for Hyper Text Markup Language.
  • HTML is the standard coding language for making/creating Web pages.
  • HTML details the composition of a Web page.
  • HTML is made up of a series of elements.
  • HTML elements designate pieces of content like "this is the paragraph", "this is a heading", "this is the clickable link".
  • HTML elements inform the browser how to display content on the users screen.

What are the elements used in this example?

  • The < !DOCTYPE html > declaration specifies that this document is an HTML5 document
  • The < html > element is the root element of this HTML page
  • The < head > element contains meta information like scaling and stylesheet links
  • The < title > element specifies a title for the HTML page which is shown on the tab bar in a browser.
  • The < body > element specifies the document's body, and is also container for all the content you can see, such as headings, paragraphs, slideshows, images, links, etc.
  • The < h1 > element specifies a heading and its size
  • The < p > element specifies a paragraph


What are HTML elements?

HTML elements are made with a staring tag, element contents, and the ending tag e.g:

< exampletag > Element Contents < /exampletag >

An HTML element is all things from the start tag to the end tag, although there are some exceptions like empty elements which do not have an end tag.
