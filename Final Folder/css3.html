How to Set up a border



This lesson will be about

  1. Types of Border
  2. Changing size of a Border
  3. Changing color of a Border


1.Types of Border style



There are

  • No border
  • Solid border
  • Dashed border
  • Ridge border
  • Grove border
  • Inset border
  • Outset border
  • Hidden border
  • Mixed border
  • Dotted border

Ways to code border.

  • p{border:solid}
  • p{border-style: solid}
  • < p style="border:solid" > inline



    • 2.Changing Border width



    Border width can be edited by

    • Putting a number PX front of the border style(e.g border:10px solid)
    • Setting the size in the, Border-width: number (or words e.g thin)


    3.Changing Border color



    Changing border color is very simular to changing size(width))

    • Putting a color name(or rgb or hex ) front of the border style (e.g border:10px red solid)or(border:red 10px solid )
    • Setting the color in the, Border-color:color name (or rgb or hex )
